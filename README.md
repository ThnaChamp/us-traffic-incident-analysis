# 🚦 Predicting Traffic Incident Duration and Impact Distance
**Project for Analyzing and Predicting US Accident Data**

## 📝 Overview
โปรเจ็กต์นี้มีวัตถุประสงค์เพื่อวิเคราะห์ปัจจัยที่มีผลต่อการเกิดอุบัติเหตุบนท้องถนนในสหรัฐอเมริกา และสร้างโมเดล Machine Learning เพื่อทำนาย:
1. **Duration:** ระยะเวลาที่การจราจรติดขัดหลังจากเกิดอุบัติเหตุ
2. **Impact Distance:** ระยะทางที่ได้รับผลกระทบจากการติดขัด

ข้อมูลที่ใช้ครอบคลุมการเกิดอุบัติเหตุในระดับประเทศ (US-Accidents Data) ซึ่งประกอบด้วยข้อมูลสภาพอากาศ, ช่วงเวลา, และลักษณะของถนน
